# 🤖 冷静投 (CalmInvest) —— AI 智能个人投资者交易纠偏助手

“冷静投”是一个专注于行为金融学纠偏的 AI 智能体项目。它专门在个人投资者冲动交易、做决策前执行心理与逻辑上的纠偏，帮助投资者避开非理性深渊。

---

## 🌟 核心亮点功能

### 1. 📊 交易前行为偏误雷达
当用户输入交易想法后，系统将自动从**11种常见投资者行为学偏误**（外推偏差、处置效应、有限注意、过度自信、羊群效应、博彩偏好、损失厌恶、证实偏差、组合忽视、心理账户、锚定效应）中精准识别并打分。前端将拦截评分，并以**直观的可视化彩色进度条仪表盘**呈现给用户。

### 2. 🎛️ 动态交易参数工作台
界面提供可弹性增减的参数输入模块。用户可以：
- 自定义添加拟操作的多行证券代码，选择买入/卖出/加仓/减仓操作。
- 输入拟操作数量与可承受亏损比例。
- 直接粘贴持仓表，系统会自动解析提取其中包含的股票代码进行联合分析。

### 3. 📰 真实行情与新闻数据增强
系统拒绝脱离现实的空谈！在诊断发起时，后台会并发请求真实接口，抓取当前股票的**最新全网实时行情**以及来自 Google News 的**近日相关新闻速览**，将其无缝作为 AI 的环境背景参数。

### 4. 🚨 强制反悔与两级阻断机制
为真正起到“冷静”效果，项目设计了严密的交互闭环：
- **反方思考 (Consider-the-opposite)**：AI 会强制列出至少 2 条与用户预期相反的市场可能，打破证实偏差。
- **两级风险弹窗**：若用户在风险提示后仍选择“执意交易”，系统会触发**最高级别红色警告**。若仍坚持，该操作将被打上负面非理性标签强制归档至个人复盘库。

### 5. 🔑 完善的本地状态管理
- 纯前端轻量化方案，基于 `localStorage` 实现多账户独立登录管理[cite: 1]。
- 自动隔离不同账号的会话历史，并在重新登录时恢复历史复盘配置[cite: 1]。

## 🛠️ 技术栈

- **核心大脑**：DeepSeek 大模型 (`deepseek-chat`)
- **交互前端**：纯原生 JavaScript (Vanilla JS) + HTML5 + CSS3
- **数据源增强**：腾讯财经行情接口 + Google News RSS 流
